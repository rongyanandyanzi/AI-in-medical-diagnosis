# Safety Guidance

Medical phone photos are often incomplete. A safe assistant must avoid turning visual uncertainty into confident medical claims.

## Core Rules

- Prefer formal report text over scan screenshots.
- Treat CT/MRI/ultrasound phone photos as incomplete unless original DICOM or full report is available.
- Explain visible findings and report terms, but do not diagnose.
- Use uncertainty language when quality or context is limited.
- Ask for clinician review for clinically meaningful decisions.

## Red Lines

Do not say:

- "This is cancer" or "This is not cancer" from a photo.
- "No bleeding/fracture/infarct/embolism" from a photo.
- "This is benign/malignant" from a screenshot.
- "You do/don't need treatment" from a single report photo.

## Safer Alternatives

Say:

- "The visible report text says..."
- "This value is above the reference range shown in the screenshot."
- "From this phone photo alone, I cannot determine..."
- "A formal radiology report or original DICOM would be needed."
- "These are good questions to ask your doctor..."

## Urgent Symptoms

If the user mentions severe chest pain, stroke-like symptoms, severe breathing difficulty, severe trauma, heavy bleeding, new weakness/numbness, altered consciousness, or other emergency symptoms, advise urgent medical evaluation instead of continuing report interpretation.
